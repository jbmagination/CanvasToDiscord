# Compatibility List
## Built-in Canvas features
The following **WILL** display as expected:
- Regular text
- Bold text
- Italics text
- Strikethrough text
- Horizontal lines

The following **WILL NOT** display, but has a fallback:
- Tables
- Images
- Math equations
- Most embeds

## Apps
Anything not on this list is untested.

The following **WILL** display as expected:
- Commons Favorites
- Google Hangouts Meet
